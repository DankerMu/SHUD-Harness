# Patch: `.gitmodules` 格式检查

**目标文件：** `.gitmodules`  
**合并方式：** 若当前文件在 raw 视图或本地解析中为单行，应替换为标准多行 git-config 格式。

## 建议内容

```ini
[submodule "SHUD"]
    path = SHUD
    url = https://github.com/SHUD-System/SHUD.git

[submodule "rSHUD"]
    path = rSHUD
    url = https://github.com/SHUD-System/rSHUD.git

[submodule "AutoSHUD"]
    path = AutoSHUD
    url = https://github.com/SHUD-System/AutoSHUD.git

[submodule "zero"]
    path = zero
    url = https://github.com/V1ki/zero.git
```

## 验证命令

```bash
git config --file .gitmodules --get-regexp '^submodule\..*\.(path|url)$'
git submodule sync --recursive
git submodule update --init --recursive
```

## 验收标准

- [ ] 本地 `git config --file .gitmodules --get-regexp` 能列出 8 行 path/url。
- [ ] CI checkout submodules 成功。
